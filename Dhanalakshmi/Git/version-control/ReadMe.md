1. git --version -> to check git is installed or not and also which version of git
2. mkdir  -> to create the sample folder
3.cd sample folder -> to navigate to that particular repo
4.git init -> initialized the git. when we run this command it will create .git folder
5. git config --global user.email "" -> used to set the gmail
6. git config --global user.name "" -> used to set the username . These two are required to push the commit and push the code to our remote git repo from local git repo.
7.git config --list -> to view the username and email set or not
8.git status -> defines the untracked/tracked files to add 
9.git Commit -m "meaning full message"
10.vim filename -> to create the file
11. cat filename -> to view the content in the file
12. git diff -> to view the difference like what modified from the last commit
13. git log --oneline -> to view the history
14. git checkout commitId -> to view the particular version
15. git checkout main  -> to switch to the main branch.
